如果你对荣格心理学感兴趣，那么继续往下了解大脑和神经科学会很有意思。因为荣格讨论的是“心理现象”，而神经科学研究的是这些现象背后的“生物硬件”。

可以把大脑想象成一座拥有约860亿个神经元的超级城市。

---

# 一、神经元（Neuron）

神经元是神经系统最基本的工作单位。

你可以把它理解成：

> 大脑里的一个“信息处理节点”。

就像互联网中的一台计算机。

一个典型神经元长这样：

```
树突 → 细胞体 → 轴突 → 突触
```

### 1. 树突（Dendrite）

负责接收信息。

像树枝一样向外延伸。

```
      \ | /
       \|/
      [O]
```

其他神经元发来的信号，大多数都先到达树突。

作用：

* 接收信息
* 汇总信息
* 决定是否继续传递

---

### 2. 细胞体（Soma）

神经元的“中央处理器”。

里面有：

* 细胞核
* DNA
* 各种细胞器

负责：

* 维持生命活动
* 处理接收到的信息

---

### 3. 轴突（Axon）

神经元的“电缆”。

长度差异极大：

* 短的几微米
* 长的可超过1米

例如从脊髓到脚趾的运动神经。

```
[细胞体]==================
```

负责把信息发送出去。

---

### 4. 髓鞘（Myelin）

包裹轴突的绝缘层。

类似：

> 电线外面的塑料皮。

作用：

* 防止漏电
* 提高传导速度

有髓鞘：

```
----()----()----()----
```

没有髓鞘：

```
----------------------
```

速度可能相差几十倍甚至上百倍。

---

# 二、神经冲动（Action Potential）

神经元之间传递信息不是靠文字。

而是靠：

> 电信号

称为：

**动作电位（Action Potential）**

---

神经元平时带负电：

约

```
-70mV
```

称为：

**静息电位**

---

当刺激达到阈值：

例如：

* 看到食物
* 听到声音
* 被针扎一下

神经元突然放电：

```
-70mV
↓
+30mV
↓
-70mV
```

形成一个电脉冲。

这就是动作电位。

---

特点：

### 全或无原则

要么发射：

```
100%
```

要么：

```
0%
```

不存在：

```
发射50%
```

这种状态。

---

# 三、突触（Synapse）

这是最重要的概念之一。

神经元彼此并不直接接触。

中间有一条极小缝隙：

约20~40纳米。

叫：

> 突触间隙

```
神经元A  | | 神经元B
```

---

那么信息怎么过去？

答案：

### 化学信号

神经元A释放神经递质。

神经元B接收神经递质。

---

# 四、神经递质（Neurotransmitter）

可以理解为：

> 神经元之间传递消息的“信使”。

目前已发现上百种。

常见的几个：

---

## 多巴胺（Dopamine）

最出名。

很多人误以为：

> 快乐激素

其实不准确。

更准确地说：

> 奖励预测系统

负责：

* 动机
* 学习
* 成就感
* 期待感

例如：

* 游戏升级
* 刷短视频
* 赌博
* 获得奖金

都会刺激多巴胺。

---

## 血清素（Serotonin）

与：

* 情绪稳定
* 满足感
* 安全感

有关。

许多抗抑郁药都作用于血清素系统。

---

## 去甲肾上腺素

负责：

* 警觉
* 注意力
* 应激反应

例如：

突然听到巨响：

```
砰！
```

马上精神起来。

就是它在工作。

---

## GABA

大脑的“刹车”。

作用：

* 镇静
* 抑制兴奋

很多安眠药作用于GABA系统。

---

## 谷氨酸

大脑最主要的“油门”。

负责：

* 学习
* 记忆
* 兴奋传递

与GABA形成平衡。

---

# 五、神经网络

单个神经元能力有限。

但连接起来就不同了。

一个神经元可能连接：

```
1000
甚至
10000
个神经元
```

形成庞大的网络。

---

粗略估计：

大脑连接数量：

```
100万亿级别
```

以上。

这就是你产生：

* 思考
* 记忆
* 想象
* 梦境
* 自我意识

的基础。

---

# 六、学习是怎么发生的？

很多人以为：

> 学习 = 往脑子里存东西

实际上：

学习主要是：

> 改变神经元之间的连接强度

---

例如：

第一次看到Python：

```
Python
↓
陌生
```

连接很弱。

---

学习100小时后：

```
Python
↓
熟悉
↓
自动联想
↓
编码
```

连接越来越强。

---

这叫：

### 突触可塑性（Synaptic Plasticity）

神经科学最重要的概念之一。

可以概括为：

> 一起激活的神经元，会连接得更紧密。

英文：

> Neurons that fire together, wire together.

---

# 七、记忆是什么？

目前主流观点认为：

记忆并不是存放在某个“文件夹”。

而是：

> 分布在大量神经元连接模式中。

类似：

```
硬盘文件
↓
不是

整个网络结构
↓
是
```

---

例如：

“你家的橘猫”

可能同时激活：

* 视觉神经元
* 气味记忆
* 情感记忆
* 名字记忆

这些神经元一起形成一个网络。

这个网络就是记忆。

---

# 八、镜像神经元（Mirror Neuron）

这是一个很有趣的概念。

当你看到别人做动作时：

```
别人吃东西
```

你的某些神经元也会被激活。

仿佛：

```
你自己在吃
```

一样。

---

有人认为它与：

* 模仿学习
* 共情能力
* 社会行为

有关。

不过其具体作用目前仍有争议。

# 十、意识从哪里来？

这是目前科学界最大的未解之谜之一。

神经科学已经知道：

* 神经元如何放电
* 突触如何工作
* 记忆如何形成

但仍然不知道：

> 为什么这些神经活动会产生“我正在体验”的主观感受。

例如：

你看到橘猫。

科学可以解释：

* 光线进入眼睛
* 神经元放电
* 视觉皮层处理

但解释不了：

> 为什么会产生“看到橘猫的感觉”。

这被称为：

Hard Problem of Consciousness

是现代神经科学、哲学和人工智能共同面对的问题。

---

如果把整个体系用一句话概括：

**神经元是计算节点，突触是连接线路，神经递质是通信协议，神经网络是操作系统，而人的思想、人格、记忆和意识，则是这个庞大系统涌现出来的结果。**

而且有趣的是，从你前面开始关注荣格心理学来看，你现在正好站在两条研究路径的交叉点：

* 荣格：研究“心理内容是什么”（原型、无意识、人格等）
* 神经科学：研究“这些心理内容如何在大脑中实现”

这也是当代认知科学最活跃的领域之一。
