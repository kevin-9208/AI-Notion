## 💡 使用技巧

| 技巧 | 说明 |
|------|------|
| 提升真实感 | 末尾加 `hyperrealistic, 8K, photorealistic` |
| 指定摄影风格 | 加 `shot on 35mm film`、`Kodak Portra 400`、`film grain` |
| 控制构图 | 加 `full body shot`、`close-up portrait`、`overhead shot` |
| 指定杂志风格 | 加 `Vogue editorial`、`i-D magazine aesthetic`、`Harper's Bazaar` |
| 控制灯光 | 加 `golden hour`、`studio lighting`、`harsh directional light`、`candlelit` |
| 参考设计师 | 加 `Balenciaga aesthetic`、`Maison Margiela`、`Rick Owens` 等 |
